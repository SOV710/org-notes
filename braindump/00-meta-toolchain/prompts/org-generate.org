1. 标题生成:
   - 自行根据内容，拟定英文标题

2. 标点符号规范:
   - 中文句号: 使用全角 (。)
   - 其他标点: 统一使用半角 (, ! ? : ; etc.)
   - 半角标点前后加空格 (符合英文习惯)
   - 示例: "这样 (对的) 和, 这样! 正确."

3. Org-mode 语法:
   - 粗体: *文本* (单星号, 不同于 Markdown 的 **)
   - 斜体: /文本/ (斜杠)
   - 下划线: _文本_ (下划线)
   - 删除线: +文本+ (加号)
   - Latex公式: $文本$ (美元符号)
   - 行内代码: =code= (等号, 而不是反引号)
   - 行内逐字文本: ~text~ (tilde, 常用于不高亮的字面量)
   - 列表:
     - 无序列表: - 项目 (短横线 + 空格)
     - 有序列表: 1. 项目
   - 标题:
     - * 一级标题
     - ** 二级标题
     - *** 三级标题
   - 链接:
     - 普通链接: [[https://example.com]]
     - 带描述链接: [[https://example.com][描述文本]]
   - 本地文件 / 图片引用:
     - [[file:path/to/file]]
   - 引用块:
     - #+begin_quote
       引用内容
       #+end_quote
   - 代码块:
     - #+begin_src language
       code
       #+end_src
   - 长横线:
     - -----

4. 文件结构:
   - 包含完整 header (#+title, #+startup)
   - #+startup: showall
   - ***禁止调用生成文件编辑 (edit file) 工具，禁止输出文件***

5. 额外条件：
   - 禁止在表格中写 latex 表达式
     如果要写 latex 表达式，就不要使用表格格式
   - 禁止在 latex 表达式中写中文
