* 键轴的物理结构

标准 MX 式机械键轴由五个核心部件构成 : *上盖* (top housing), *轴心* (stem), *弹片* (contact leaf / crosspoint), *弹簧* (spring), *下底* (bottom housing)。所有面向消费者的轴体参数, 都可以且应当还原为对这五个部件的物理描述。若一款产品仅声称其使用效果而完全回避作用机理, 通常意味着该产品在结构层面上缺乏差异化竞争力。

** 上盖 (Top Housing)

上盖承担两个功能 : 为轴心提供竖向导向, 以及构成整体键轴的封装外壳上半段。导向精度直接影响轴心在非正向按压 ( 斜向施力 ) 时的稳定性 ; 上盖材质则深刻影响声音频率特性。

- *聚碳酸酯* (PC / Polycarbonate): 密度较低, 声音偏高频, 透光性好, 常见于追求清亮音色的轴体
- *尼龙* (PA / Nylon): 阻尼感更强, 声音偏中低频, 手感偏"肉"
- *POM*: 自润滑性能好, 摩擦系数低, 可改善轴心运行顺滑度

上盖与下底之间的装配公差 (housing tolerance) 决定轴体的"晃动量" (wobble)。公差过大时, 可以通过加装 *轴体薄膜* (switch film) 来收紧配合、改善音色一致性。

** 轴心 (Stem)

轴心是直接与键帽接触、承受按压力并在运动中触发弹片的部件, 是决定手感类型的最关键元素。

轴心的两条腿 (stem legs) 形状决定了手感类型 :

- *线性* (Linear): 两腿光滑, 对弹片施力过程连续均匀, 无段落感
- *段落* (Tactile): 两腿具有凸起 (bump), 经过弹片时产生可感知的阻力峰值, 即"段落感"
- *有声段落* (Clicky): 额外引入 click bar 或 click jacket 等机构, 在段落点同时发出声响

轴心的长度 (pole length) 影响触底时发出的声音 : 较长的轴心杆 (long pole) 触底时声音更为尖锐、明确 ; 较短的则触底更"沉"。此外, 轴心与上盖中央孔的间隙也会影响按压时的轴向晃动。

** 弹片 (Contact Leaf)

弹片是实现电气接通的机构, 由两片金属簧片组成。静态时两片分离 (断路) ; 轴心按下, 轴心腿撑开小簧片并使两片接触, 完成一次信号触发。

弹片形状、刚度以及两片之间的间距, 共同决定了 :

- 触发点的精准度
- 段落手感的强弱与陡峭程度 (bump sharpness / bump strength)
- 弹片回弹时产生的"弹片咔哒声" (leaf tick), 即在轴心回程时弹片恢复原位时的金属音

弹片润滑与否, 也会对上述音色产生显著影响 (详见润滑部分)。

** 下底 (Bottom Housing)

下底固定弹片与弹簧, 并通过金属引脚焊接或热插拔连接至 PCB。下底材质同样影响声音与共振特性, 且与上盖材质的组合搭配产生的声学效果往往是非线性的。

- 下底若配备 *防尘壁* (dustproof stem / box-style housing): 轴心被额外的方形护壁包裹, 防止异物进入同时改变声腔共振特性 (详见声音部分)
- PCB mount 轴体拥有五根引脚 (两金属 + 三塑料定位), 相比 plate mount (三引脚) 固定更牢固, 但需配合相应 PCB

* 轴体的输出空间

评估一款轴体的用户体验, 可以分解为两个正交维度 : *手感* 与 *声音*​。两者均受上述五个物理部件的综合影响。

** 手感的构成要素

手感并非单一指标, 而是多个子维度的叠加 :

1. *触底手感* (bottom-out feel): 轴心运行到底部极限时传递给手指的反馈质感 (详见游戏轴章节)
2. *按压压力曲线* (force curve): 全程按压力随行程的变化规律, 由弹簧与弹片共同决定
3. *段落感* (tactility): 有无阻力凸起, 以及凸起的强度、位置、陡峭程度
4. *轴向顺滑度* (smoothness): 轴心在运动导轨上的摩擦感, 受材质与润滑影响
5. *回弹速度* (return speed): 轴体恢复初始位置的速度, 受弹簧参数影响
6. *上下限位松紧* (stem-to-housing tolerance at travel limits): 轴心在最高点与最低点的晃动量

** 声音的构成要素

声音的影响因素更为分散, 涵盖轴体内部与外部结构 :

- 轴体材质组合 (上盖 + 下底的材质搭配)
- 防尘壁 / box 结构对声腔的影响
- 润滑量与润滑分布
- 轴心杆长度 (触底音色)
- 弹片回弹产生的金属音
- 弹簧 ping 声 (弹簧与中柱摩擦或弹簧两端接触结构产生的共鸣)
- 键盘外部结构 : 定位板材质、安装方式 (gasket / top mount / tray mount)、PCB 厚度等

* 游戏轴的选型逻辑

** 游戏轴的三个核心诉求

市场上明确标注"游戏轴"定位的产品, 通常围绕以下三个核心卖点构建 :

1. *手感稳* : 轴体在高速、反复按压时保持一致的触发感受, 不出现"飘感"或不一致触发
2. *触底扎实* : 触底手感明确, 给予使用者清晰的操作确认反馈
3. *压力偏重* : 相比打字轴略重的弹簧克数, 减少误触概率, 提升操作精准度

游戏轴的线性特征也是普遍共识 : 无段落的线性轴在快速连击时不存在段落阻力带来的疲劳感, 且触发曲线更加可预期。

** 触底手感分类 : 脆轴与闷轴

触底手感是区分不同游戏轴 (乃至所有线性轴) 的最直接维度, 本质上由 *下底材质硬度* 与 *轴心杆末端的结构* 共同决定。

*** 脆轴 (Crispy Bottom-out)

脆轴的触底特征为 : 触底瞬间有清晰、干脆的硬质反馈, 类似"咔"的质感。成因在于 :

- 下底使用硬度较高的材质 (如 POM 或高硬度 PA)
- 上下限位配合较紧, 几乎没有缓冲行程
- 轴心在到达底部时传递的冲击力被清晰地反射回手指

脆轴可进一步细分 :

- *标准脆轴* : 触底音频偏高, 手感冲击感强, 典型代表如传统 POM 下底轴体
- *中频脆轴* : 触底音频介于高低之间, 触底反馈同样明确但声音稍沉, 多为上盖选用 PC、下底选用 Nylon 的组合

市面上绝大多数自我定位为"游戏轴"的产品均属于脆轴范畴, 因为其明确的触底反馈与"扎实感"更契合游戏操作的确认需求。

*** 闷轴 (Thuddy Bottom-out)

闷轴的触底特征为 : 到底时冲击力被吸收, 手感更"沉", 没有明显的硬质弹力感。典型成因 :

- 下底材质使用相对较软或阻尼较高的材料
- 上下限位存在一定弹性缓冲

闷轴同样可细分 :

- *闷响轴* : 触底仍有一定低频声音反馈, 材质较脆轴软但尚有结构响应 ; 整体手感偏"肉"而非完全无感
- *纯闷轴* : 触底几乎无声、无明显手感反弹, 极度安静 ; 此类轴体通常在轴心上集成硅胶阻尼垫 (silencing pad), 即所谓静音轴

#+begin_quote
需要注意的是 : 脆轴/闷轴的分类并非绝对的好坏判断, 而是使用场景与个人偏好的匹配问题。对于游戏玩家而言, 脆轴的明确触底反馈通常更受欢迎 ; 对于深夜办公或需要低噪音的场景, 闷轴或静音轴更为适宜。
#+end_quote

* 润滑对手感与声音的影响

润滑 (lubrication) 是轴体调教中最常见、效果最显著的变量之一, 直接影响顺滑度、声音频率特性与弹片音色。

** 润滑量的影响

- *润滑量偏多* : 轴心运动阻力降低, 整体手感偏"腻" (mushy), 压力曲线感知趋于模糊 ; 过量润滑可能抹平线性轴的顺滑感上限, 同时导致触底手感失去"干脆"的边界感
- *润滑量适中* : 顺滑度提升而不丧失手感层次感 ; 此为大多数轴体调教的目标状态
- *润滑量偏少或未润滑* : 手感偏"干爽" (dry), 轴心运行摩擦感更明显 ; 弹片弹回时的金属咔哒声 (leaf tick) 明显加重 ; 同时弹簧若未润滑, 在弹簧两端与结构件接触时容易产生"ping"声 (弹簧共振)

** 润滑区域的差异

不同部位的润滑对应不同效果 :

| 润滑区域     | 主要效果                                                      |
|-------------+--------------------------------------------------------------|
| 轴心腿侧面    | 降低运行摩擦, 提升顺滑度                                         |
| 上下盖内壁导轨 | 减少轴心晃动摩擦                                                |
| 弹簧两端     | 消除 ping 声                                                  |
| 弹片        | 降低 leaf tick 音量 (谨慎 : 段落轴/有声轴不建议润滑弹片, 否则手感失真) |

#+begin_quote
对于有声轴 (clicky switches), 润滑轴心腿或弹片会抑制甚至消除 click 声响, 通常只建议润滑弹簧部分。
#+end_quote

* 声音参数详解

** 防尘壁对声音的影响

防尘壁 (dustproof stem / box-style stem housing) 是指在下底对轴心进行四面包裹的方形护壁结构 (以 Kailh Box 系列为代表)。其对声音的影响机制如下 :

- 防尘壁改变了轴心运行时的声腔空间 : 传统 MX 轴心四周是相对开放的空腔, 而 Box 结构将轴心包裹其中, 形成封闭或半封闭腔体
- 封闭腔体导致共振路径改变, 通常使触底声音频率降低、更为"厚实"
- 防尘壁同时减少轴心的横向晃动, 从而改善击键一致性
- 轴心与防尘壁之间的间隙公差也会影响运行时的摩擦音

此外, 键盘整体系统的声学特性 (定位板材质、安装方式等) 同样对最终听感产生决定性影响, 轴体本身的声音特性只是整个声学系统的一个环节。

** 声音频率的常见描述

| 描述词 | 特征 | 典型成因 |
|--------|------|----------|
| Clacky | 高频、清脆 | PC 上盖、铝定位板 |
| Thocky | 中低频、厚实 | Nylon 下底、黄铜定位板、gasket 安装 |
| Poppy | 高频尖锐 | 长轴心杆触底、PC 全壳 |
| Thud | 极低频、几乎无声 | 静音轴、硅胶阻尼垫 |

* 弹簧参数详解

弹簧是决定按压压力曲线的核心部件, 其参数对手感的影响往往比更换轴心或外壳更为直接。

** 压力克数 (Spring Weight)

压力克数是消费者最常接触的弹簧参数, 通常标注为 *触发力* (actuation force) 或 *触底力* (bottom-out force)。

- 人手在自然放松状态下的手指落键力约为 $40\ \mathrm{g}$ 左右, 因此 $40\ \mathrm{g}$ 以下的触底力基本处于"不需要主动施力"的区间
- 市面上常见克数分布 :
  - 轻弹簧 : $35\ \mathrm{g}$ – $45\ \mathrm{g}$ 触底, 适合打字速度快、长时间使用、追求轻盈感的用户
  - 中等弹簧 : $50\ \mathrm{g}$ – $65\ \mathrm{g}$ 触底, 兼顾误触防护与打字舒适
  - 重弹簧 : $67\ \mathrm{g}$ 以上, 常见于游戏轴 ; 反馈扎实、误触率低, 但长时间使用手指易疲劳

#+begin_quote
注意 : "触发力"与"触底力"存在差异。触发力是弹片刚完成电气接通瞬间所需的力, 受弹片弹力影响 ; 触底力是轴心行程终点所需的力, 完全由弹簧刚度决定。同一弹簧配合不同弹片, 触发力可能不同。更可靠的参考指标是触底力。
#+end_quote

压力克数的个性化差异极其显著, 取决于用户的手指力量、打字习惯、使用时长以及对误触的容忍度。没有绝对优劣之分, 建议通过轴体测试板 (switch tester) 亲身测试后决定。

** 弹簧长度

弹簧的自然长度决定了力曲线的斜率特性 :

- *标准长度弹簧* (约 $14\ \mathrm{mm}$): 按压初段力度较轻, 尾段压力增长较快 ; 力曲线斜率较陡
- *长弹簧* (约 $20\ \mathrm{mm}$ 以上): 相同触底克数下, 初段力度更接近触底力 ; 力曲线更平缓 (flatter force curve)。具体表现为 : 整个行程中手指感受到的力变化幅度更小, 给人"压力均匀"的感知

选用长弹簧时需注意 : 由于物理长度超过轴体内腔, 弹簧可能与中柱 (center mast) 摩擦产生噪音, 建议在中柱接触段涂抹润滑脂。

** 单段弹簧 vs. 双段弹簧 (Multi-stage Springs)

*** 单段弹簧 (Single-stage Spring)

结构为均匀螺距的标准螺旋弹簧, 弹力随形变线性增长 (即 Hooke 定律 $F = kx$, 其中 $k$ 为常数)。

- 尾程压力增长明显 : 由于弹簧从初始位置到完全压缩的形变量较大, 触底时相对初段会有可感知的压力累积
- 手感预期感强 : 按压过程中力的增长规律性好, 适合需要稳定可预期手感的用户

*** 双段弹簧 (Two-stage / Dual-stage Spring)

双段弹簧在单根弹簧内引入两个不同的螺距区段 (variable pitch coiling), 紧密绕制段与稀疏绕制段的刚度不同, 从而实现分段式弹力特性 :

- 按压前半段 : 稀疏绕制段先被压缩, 弹力增长相对缓慢 (较轻)
- 按压后半段 : 密集绕制段介入, 弹力增长加速 (渐重)
- 整体效果 : 产生"渐进式" (progressive) 压力曲线, 回弹速度相对更快 ; 轴心复位力更集中在行程末段

双段弹簧的实际感知效果 :

- 初段轻盈, 减少长时间打字的手指疲劳
- 末段有力, 回弹快速, 为下一次击键提供更清晰的"归位感"
- 对于段落轴而言, 双段弹簧的弹力曲线有时能强化或改变段落手感的感知

#+begin_quote
三段弹簧 (three-stage spring) 是双段弹簧的延伸设计, 主要用于更长的弹簧以抑制受压弯曲失稳, 在提供更细腻的渐进曲线之外, 稳定性也优于两段设计。
#+end_quote

* 打字轴 vs. 游戏轴 : 选型对比小结

打字轴 (typing switches) 与游戏轴在核心诉求上存在结构性差异 :

- *打字轴*: 更注重轻压力启动 + 良好回弹 ; 段落感可帮助触觉确认触发点, 减少视觉依赖 ; 弹簧克数偏轻 ; 对触底手感的"扎实性"要求低于游戏轴
- *游戏轴*: 注重线性无阻手感、稳定一致的触发、较重压力防误触, 以及扎实的触底反馈

最终的选型应将上述参数作为框架, 结合具体使用场景 (FPS / MOBA / 长文本输入 / 混合使用) 与个人生理特征 (手指力度、敏感度偏好) 进行匹配, 而非依赖营销话术。
