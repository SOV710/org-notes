* Scheme Characteristics

虎码为 26 键、四码上限的乱序形码方案, 具备以下核心特性 :

#+begin_quote
很多人熟悉的五笔方案, 是 25 键方案, 没有 Z 键。
#+end_quote

- *双编码* : 每个字根对应 大码 + 小码 两个字母
- *字根归并* : 同源或形近字根合并为主根 + 变体, 实际记忆量为 241 个主根
- *大字根* : 拆分粒度粗, 直观, 少拆笔画
- *字集支持* : UTF-8 全汉字集, 近 10 万字

性能方面 : 前 1500 字全码 0 重码; 前 3000 字出简后 0 重码; 国标一级字 (3500 字) 加内置回头码后 0 重码。

* Dual Encoding: 大码与小码

字根所在键位即为 *大码*​。

*小码* 为音托规则 :
- 大部分取声母; 少量取韵母
- 读音为 yi/wu/yu 的字根, 小码取 =i= / =u= / =v=
- 声母为 z 的字根取韵母 (降低 =z= 键使用率), 但 "走 =pz=" 仍取声母, 因 =po= 难按

#+begin_quote
音托并非为了辅助记忆字根, 而是在 *不确定小码时的应急手段*​。
若将小码直接当作两个字母强记, 上手往往更快。
#+end_quote

* Encoding Rules

** 单字取码

规则一句话概括 : *取前 3 根 + 末根的大码 ; 不足四码则补末根小码*​。

| 根数         | 取码模式 | 示例                                  |
|--------------+---------+---------------------------------------|
| 1 根 (字根字) | Aa      | 高 =Gg=, 九 =Kj=                          |
| 2 根         | ABb     | 难 : 又 Ry + 隹 Ui → =RUi=               |
| 3 根         | ABCc    | 华 : 亻Jr + 匕Vb + 十Ns → =JVNs=         |
| 4 根         | ABCD    | 赝 : 厂Xc + 亻Jr + 隹Ui + 贝Ob → =XJUO=   |
| ≥5 根        | ABCZ    | 颦 (6 根) → =SSWN= (前 3 根大码 + 末根大码) |

*字根字让位规则*: 部分字根字会让位给高频字的二简, 退至次选位。此类字可重复末码后按空格输出, 例如: 己 =vjj=, 鱼 =wvv=, 骨 =fgg=​。

** 词组取码

| 词长    | 规则                   | 示例                      |
|---------+------------------------+---------------------------|
| 二字词   | 各取前 2 码             | 秃头 : =xo= + =wm= → =xowm=     |
| 三字词   | 前 2 字首码 + 末字前 2 码 | 针不戳 : =z= + =c= + =zu= → =zczu= |
| 四字词   | 各取首码                | 风风火火 → =oocc=            |
| ≥5 字词 | 前 3 字首码 + 末字首码    | 中华人民共和国 → =djjr=        |

** 简词 (进阶, 新手跳过)

简词为高频口水词的缩略打法, 属可选项, 不影响全码词打法。

- *一简词*: 词的第一码 (可为多字词), 如 =w= 怎么, =i= 为什么
- *二简词*: 取两字首码 ; 双字母次选位为叠词/同部首词, 如 =kk= 渐渐, =oo= 尴尬
- *三简词*: 取三字首码 (占空码位), 如 =rcp= 对不起
- 无四简词: 四字词全码即为各字首码

* Decomposition Principles

** 基本拆分顺序

从上到下, 从左到右。走字底、建字底等后写部件符合笔顺, 后取 :

- 过 → 寸 + 辶 ; 廷 → 丿 + 土 + 廴

** 字架优先

含包围结构的字, 优先以外框 (字架) 为第一字根, 内部内容后取 :

- 哀 → 衣 (字架) + 口 ; 街 → 行 (字架) + 圭

** 切字 (笔画剪断)

糅合度高的字根必要时允许切断笔画, 通常仅切一次 :

- 果 → 田 + 木 (丨被横向剪断)
- 丝 → 纟 + 纟 (一被垂直剪断)
- 惠 → 十 + 田 + 心 (切两次, 特例, 见图四)

带 𤰔 的字统一拆为 十 + 田, 如 : 叀, 恵。

** 形似根同键 (大码相同)

外形相近的字根置于同一键位, 降低记忆负担, 提高看打容错 :

- =G= : 士, 土
- =V= : 己, 已, 巳
- =U= : 廴, 辶
- =L= : 艹, 廿, 卅, 卌
- =H= : 夂 (右不出头, 多在上下), 攵 (右出头, 多在右部)
- =X= : 鸟, 乌 ; 广, 厂
- =K= : 母, 毋
- =A= : 乛, 乙

** 口 与 囗 的区别

- 口 =Dk= : 内部无内容, 如 : 叫, 喊
- 囗 =Rk= : 内部有内容, 如 : 国, 回

框结构按开口方向区分 :

- 冂 =Mk= : 开口在上或下 (助记 : 瞒 Mán 上欺下 → M 为上下)
- 匚 =Nk= : 开口在左或右 (助记 : 男 Nán 左女右 → N 为左右)

* Character Set Coverage

虎码覆盖 UTF-8 全汉字集 (基本区至扩 I 区, 含笔画, 部首补充, 兼容字等), 近 10 万字。

定位原则 : 常用字重码极低, 确定性强 ; 大字集虽有选重, 但重在 *能打出来*, 而非强行以复杂规则避重。
