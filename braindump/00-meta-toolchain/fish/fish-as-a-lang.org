fish (Friendly Interactive SHell) 的设计哲学是开箱即用、语法清晰、避免历史包袱。我们直接从语言本身切入。

* Grammar

** Variable

#+begin_src fish
# 赋值用 set, 没有 $ 在左边
set name "world"
set count 42

# 读取用 $
echo $name
echo "hello $name"

# 列表 (fish 原生支持列表, 这是核心特性)
set fruits apple banana cherry
echo $fruits[1]        # apple (下标从 1 开始)
echo $fruits[2..]      # banana cherry (切片)
echo (count $fruits)   # 3
#+end_src

作用域修饰符:

#+begin_src fish
set -l local_var "只在当前作用域"   # local
set -g global_var "全局"            # global
set -x EXPORTED "导出为环境变量"    # export (等价于 export)
set -e var_to_delete                # 删除变量
#+end_src

注: 这是一个很大的坑点, 也是一个很大的难点, 后面我们会再讲到这个配置

** String

#+begin_src fish
# 双引号: 插值
set who "fish"
echo "hello $who"          # hello fish

# 单引号: 字面量, 无插值
echo 'hello $who'          # hello $who

# 字符串操作 (内建 string 命令, 非常强大)
string length "hello"                    # 5
string upper "hello"                     # HELLO
string replace "foo" "bar" "foo baz"     # bar baz
string match -r '\d+' "abc123"           # 123
string split "," "a,b,c"                 # a b c (列表)
string join "-" a b c                    # a-b-c
#+end_src
